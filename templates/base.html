About the project

This web application was built using python with the flask framework. We use a purpose-built web scraper to collect sales data from ebay UK. Our scraper is also built using python and stores the scraped data in a postgresql database, which is accessed remotely by this application.

This website is not affiliated with ebay in any way and is a not-for-profit hobby project.
